## The Free Ride Engine: How 73% of Skills Cost $0

This is the key innovation in OpenClaw's architecture. The **Free Ride Engine** is a routing layer that evaluates each skill call and routes it to the most cost-effective model that can handle the task.

The tiers work like this:

**Free Tier (S0 — Meta skills):** Skills that don't need AI at all — file operations, data transformation, template filling — run at exactly $0.

**Haiku Tier (S1-S2 — Research + Social):** Quick research and social media generation tasks route to **Claude 3 Haiku**, which is 25x cheaper than Sonnet. At current pricing, Haiku costs approximately $0.25 per million input tokens — making high-volume research and social generation practically free.

**Sonnet Tier (S3-S5 — Blog, Offers, Analytics):** Medium-complexity tasks like blog writing, landing page generation, and analytics processing route to **Claude 3.5 Sonnet**. This is where most of your API spend happens.

**Opus Tier (S6-S8 — Deep Analysis):** Complex strategic tasks (A/B test interpretation, self-improvement loops, comprehensive audits) use **Claude 3 Opus** selectively. These run less frequently — typically weekly — which keeps costs in check.

The routing result: 38 of 52 skills (73%) run on free or near-free models. Only 14 skills regularly use paid API capacity.
